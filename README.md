# Data-Science-III-Unidad-2-An-lisis-de-sentimiento
Procesamiento de Texto y Análisis de Sentimiento
# 🚀 Advanced NLP Lab: End-to-End Sentiment Analysis on Yelp Dataset

Este repositorio presenta un pipeline avanzado de **Procesamiento de Lenguaje Natural (NLP)** para resolver un desafío de negocio real: la clasificación automatizada de opiniones de clientes y la medición del impacto de sus errores de predicción en la operación corporativa.

## 🛠️ Mejoras Técnicas Implementadas (V2)

* **Pipeline de Preprocesamiento Profundo (spaCy):** Implementación de normalización de texto mediante tokenización, remoción de stop words y reducción a raíces semánticas (**Lematización**) para optimizar el vocabulario activo.
* **Métricas de Evaluación Integrales:** Implementación matemática explícita de matrices de confusión para auditar el balance de predicciones correctas e incorrectas.
* **Análisis Crítico de Negocio:** Vinculación de las métricas estadísticas (Falsos Positivos y Falsos Negativos) con impactos financieros reales en la retención de clientes y costos de soporte.

## 📊 Tecnologías
* Python 3.x, Pandas, spaCy (`en_core_web_sm`), TextBlob, Scikit-Learn, Seaborn, Matplotlib.
